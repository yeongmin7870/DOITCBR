1. Cobra2000 폴더 아래 
	Cobradoc.exe , 코브라유틸 설명 참조

2.  Cobra.exe 파일 실행 
cobradoc -i "spl 이라는 output data 파일 (input 경로)" -D "output 경로"   Ps (PostScript) 데이터 추출

3. Pdf 변환

	1) 고속 PDF 변환
		cobradoc -C "PDFMODE Yes"   => pdf파일, LST 파일 생김 
		Because 기존에는 PS 파일에 이미지 데이터가 같이 들어있어서 하나씩 이미지를 PDF 배경으로 변환하기에 느렸는데
		LST 파일을 만들고 경로만 선택해서 고속으로 PDF 변환 할 수 있게 됨
		ex) C:\Users\yeong>C:\cobra\COBRA2000\cobradoc.exe -i E:\연습프로젝트\결과폴더\output.txt -D E:\연습프로젝트\결과폴더
		ex) C:\Users\yeong>C:\cobra\COBRA2000\cobradoc.exe -i E:\연습프로젝트\결과폴더\output.txt -D E:\연습프로젝트\결과폴더 -J [GB 배경 삽입할지 안할지 여부] -C "PDFMODE YES" [고속 pdf 변환 여부]
		ex) CRTPDFC -i [input 경로]
	2) 일반 PDF 변환
		crtacvres 그냥 이렇게만 cmd 쳐도 관련 명령어 설명들이 나옴 
		ex) C:\Users\yeong>C:\cobra\COBRA2000\CRTACVRES.exe -i E:\연습프로젝트\결과폴더\output.dat -o E:\연습프로젝트\결과폴더\output.pdf -O pdfwrite

이외 정보

	-cobra.ini
		output 경로 설정 할 수 있음
	 c
	-배경 포함여부 
		COBRADOC -J : ps data를 배경이미지에 포함 (cobra.exe)
		-J 면은 배경이미지 포함 -J 가 없으면 배경이미지 미포함 
		이때 중요한점은 CBR 상에서 GB로 설정했을때만 사용할 수 있다. 
		Because 프린트기에 찍을때는 배경이미지를 제외하고 데이터를 찍어야 하고
		고객에게는 배경이미지와 데이터를 포함해서 보여주어야 하기 때문에
